# Acar-et-al.2022
25RMD24 佐藤が 2024-2025 に行った Acar et al., 2022 の再現実験

## プロジェクト概要
本リポジトリは、BADE(Bias Against Disconfirmatory Evidence) と陰謀論信念・疑似科学信念の関連性を調査する実験データを提供します。
この研究は、Acar et al., (2022)の研究を再現をし、より包括的なアプローチを採用しています。

### 実験1
Acar et al., 2022の実験を拡張、疑似科学信奉尺度，科学的推論尺度，認知的熟慮性，被害妄想尺度を追加した追試

### 実験2
実験1の内容に対して、別のBADE（感情的BADE）を追加、群を分けて検証

データの内容
/data
  /Experiment1_ver1        - 実験1の実験データ
  /Experiment2_ver1  - 実験2の実験データ
/analysis           - データ解析に使用したスクリプト
  /実験1.Rmd        - 実験1の分析コード
  /実験2.Rmd        - 実験2の分析コード  
/question           - 実験に使用した質問項目
  /尺度項目_調査1        - 実験1の質問項目
  /尺度項目_調査2        - 実験2の質問項目

加工済みデータに関して

以下は、上記のデータフレームの各列の意味を記載した、見やすいマークダウン形式の説明表です。README.md にそのまま貼り付けて使えます。

データフレーム列説明
列名	データの説明	データの定義域・型・取りうる値例
Participant	参加者のID	整数（例: 1, 2, ...）
Event	提示されたライフイベントのID	文字列または整数（例: 1, 2, ...）
Valence	ライフイベントに対する主観的価の評価(1~5)	整数（1, 2, 3, 4, 5）1=非常にネガティブ, 3=中立, 5=非常にポジティブ
Valence2	Valenceから計算された3つの値	カテゴリ（"Negative", "Neutral", "Positive"）
E1	最初の自己リスク推定値	数値（3以上77以下の離散値、リスク推定値）
eBR	推定された参加者による一般的な人の基準リスク	数値（3以上77以下の離散値、推定された一般的な人のリスク）
BR	実際の一般的な基準リスク	数値（0以上の連続値、実際の一般的な基準リスク）
E2	2回目の自己リスク推定値（更新後）	数値（3以上77以下の離散値、2回目の自己リスク推定値）
BRRecall	参加者が思い出した基準リスクの値	数値（3以上77以下の離散値、思い出した基準リスク）
gender	性別	カテゴリ（"M", "F"）
age	年齢	整数（例: 18, 25, 40, ...）
response_time	回答時間（秒）	数値（秒、例: 1500, 1000）
education	学歴（例：高校卒、大学卒など）	カテゴリ（例: "高校卒", "大学卒", "大学院卒" など）
marital_status	婚姻状況（例：独身、既婚など）	カテゴリ（例: Single, Married）
occupation	職業	カテゴリ（例: SelfEmployed, Employee, PartTime, Jobless, Other）
similar_task_flag	類似課題の経験有無（1: あり, 0: なし）	論理値（0, 1）
monthly_income	月収（単位は万円）	数値（例: 20, 30 など）
trouble_flag	回答中に問題があったかのフラグ（1: あった, 0: なかった）	整数（0, 1）
trouble_text	問題の内容（自由記述）	文字列（自由記述）
free_text	その他自由記述欄の内容	文字列（自由記述）
cesd_point	CES-D抑うつ傾向尺度のスコア（抑うつ傾向）	離散値（0~60）
lotr_point	楽観主義尺度（LOT-R）のスコア	離散値（0〜24）
jsom_point	状態楽観主義尺度（J-SOM）のスコア	離散値（7〜35）
crt_point	認知反省テスト（CRT）のスコア（論理的思考力）	離散値（0〜3）
assertiveness_mean	相互独立性（個の認識・主張）の平均スコア	連続値（1〜7）
independence_mean	相互独立性（独断性）の平均スコア	連続値（1〜7）
affiliation_mean	相互協調性（他者への親和・順応）の平均スコア	連続値（1〜7）
concern_mean	相互協調性（評価懸念）の平均スコア	連続値（1〜7）
Event_name	ライフイベントの内容を表すテキスト名	文字列（ライフイベントの内容、例: "ドライブに行く", "ドライアイになる" など）
